README - Editeur de grille

Objectif

Cette page explique en detail comment utiliser l'editeur de grilles. L'editeur sert a construire, modifier, importer et exporter des grilles pour le plateau.

Vue Generale

Nom / Taille
Definit l'identite de la grille et ses dimensions.
Grilles locales
Liste des grilles stockees dans le navigateur (localStorage).
Grilles (dossier)
Charge des grilles depuis le dossier `grids/` (JSON ou XLSX).
Definition
Associe une definition a un mot horizontal/vertical selectionne.
Mot magique
Permet de marquer un mot complet comme mot magique.
Import / Export
Importe ou exporte des grilles au format JSON.

Workflow Recommande

1) Cree ou charge une grille. 2) Remplis les lettres et numeros. 3) Definis les mots et leurs definitions. 4) Definis (si besoin) un mot magique. 5) Exporte en JSON.

Interaction Souris

Clic simple
Selectionne une case (lettre ou numero).
Shift + clic
Depuis la case active, selectionne un mot (meme ligne/colonne) ou un bloc rectangulaire.
Double-clic
Sur une case lettre: ajoute/retire le marquage magique de la case.

Saisie Clavier

Lettres A-Z
Ecrit une lettre dans la case active puis avance selon l'orientation.
Chiffres 0-9
Renseigne un numero de case (1 a 99).
Fleches
Deplace la case active dans la grille.
Backspace / Delete
Efface la case active, ou toute la selection (mot/bloc).

Raccourcis

Ctrl/Cmd + C
Copier la selection (bloc, mot ou case).
Ctrl/Cmd + X
Couper la selection.
Ctrl/Cmd + V
Coller a partir de la case active.
Ctrl/Cmd + A
Selectionner toute la grille.
Ctrl/Cmd + Z
Annuler (historique local de la session).
Ctrl/Cmd + Espace
Basculer l'orientation de saisie (Horizontal/Vertical).

Definitions

Pour enregistrer une definition, selectionne d'abord un mot complet (Shift + clic sur la meme ligne ou colonne), puis utilise le champ Definition et le bouton Sauver definition.

Mot Magique

Le mot magique se definit sur un mot deja selectionne. Utilise Definir mot magique pour l'activer, puis Retirer pour l'enlever.

Import / Export

Exporter JSON
Telecharge la grille courante en fichier JSON.
Importer JSON
Importe un ou plusieurs JSON dans les grilles locales.
Glisser-deposer
Tu peux glisser des fichiers JSON directement dans la page.

Notes Importantes

Le statut en bas du panneau (Pret., erreurs, confirmations) donne le retour d'action principal. Si un collage depasse la grille, l'operation est refusee. Quand tu reduis la taille, les cases hors limites sont supprimees.
