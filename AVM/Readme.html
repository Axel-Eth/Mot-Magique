🎤 À vos mots !

Jeu de mots type plateau télé – Version Web (HTML / CSS / JavaScript)

📌 Présentation

À vos mots ! est une application locale inspirée des jeux de lettres télévisés.

Elle se compose de deux parties :

  • La régie (index.html) : contrôle du jeu (choix de la grille, saisie des lettres, validation)
  • Le plateau (plateau.html) : affichage public de la grille, des définitions et des animations
⚠️ Cette application ne fonctionne pas en double-cliquant sur les fichiers HTML.
Un serveur local est indispensable.

📁 Structure du dossier

La structure doit être strictement respectée :

a-vos-mots/
│
├─ index.html
├─ plateau.html
├─ styles.css
├─ app.js
├─ plateau.js
├─ grids.json
│
├─ grids/
│   ├─ grille1.xlsx
│   └─ grille2.xlsx
│
└─ sounds/
    ├─ ambiance.mp3
    ├─ correct.mp3
    ├─ wrong.mp3
    ├─ grid_complete.mp3
    └─ timer.mp3

🗂️ Configuration des grilles (grids.json)

Ce fichier contient la liste des grilles disponibles dans le menu de la régie.

Exemple minimal :

[
  { "name": "Grille 1", "file": "grids/grille1.xlsx" },
  { "name": "Grille 2", "file": "grids/grille2.xlsx" }
]
  • name : nom affiché dans le menu
  • file : chemin vers le fichier Excel (relatif au dossier racine)

🚀 Lancer l’application (OBLIGATOIRE)

✅ Méthode recommandée : Python

1️⃣ Ouvre PowerShell

2️⃣ Place-toi dans le dossier du projet :

cd chemin\vers\a-vos-mots

3️⃣ Lance le serveur :

python -m http.server 8000

Tu dois voir :

Serving HTTP on 0.0.0.0 port 8000

4️⃣ Ouvre ton navigateur à l’adresse :

http://localhost:8000

❌ Ce qu’il ne faut PAS faire

Action Résultat
Double-cliquer sur index.html ❌ Ne fonctionne pas
URL commençant par file:/// ❌ Bloqué par le navigateur
Oublier de lancer le serveur ❌ Failed to fetch

🎮 Utilisation du jeu

1️⃣ Régie

  • Choisis une grille dans le menu
  • Clique sur « Ouvrir le plateau »

2️⃣ Plateau

  • Clique sur une case ou un numéro
  • La définition s’affiche en bas

3️⃣ Régie

  • Tape une lettre
  • Appuie sur Entrée

➡️ Toutes les occurrences sont révélées

4️⃣ Validation

  • Correcte → mot validé + animation
  • Nop → lettres non révélées grisées

🔁 Boutons utiles

Bouton Action
Reset état Cache toutes les lettres
F11 plateau Plein écran plateau
Correcte Valide le mot
Nop Mauvaise réponse

📘 Format des fichiers Excel (.xlsx)

✔️ Lettres

  • Cellule texte
  • La première lettre est utilisée

✔️ Numéros

  • Cellule numérique

✔️ Définitions

  • Commentaires Excel
  • Priorité :
    1. Commentaire du numéro
    2. Commentaire de la première lettre
    3. Message par défaut

⚠️ Erreur fréquente : Failed to fetch

Cause (99 % des cas)

  • Application ouverte sans serveur

Solution

  • Lancer le serveur
  • Ouvrir http://localhost:8000

🧪 Vérifications rapides

Ouvre ces URLs dans le navigateur :

  • http://localhost:8000/grids.json
  • http://localhost:8000/grids/

Si ça ne s’affiche pas :

  • Mauvais dossier
  • Fichier manquant
  • Serveur lancé ailleurs

🔊 Sons

  • Les fichiers .mp3 doivent être dans sounds/
  • Certains navigateurs bloquent l’audio tant qu’un clic n’a pas eu lieu
